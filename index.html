Keyboard Access Guide

This site is fully navigable by keyboard. No mouse required.

Key Action
Tab Move focus to the next interactive element
Shift + Tab Move focus to the previous interactive element
Enter / Space Activate a focused link or button
← → Arrow keys Move between navigation links when nav is focused
Escape Close this dialog

What we prioritise

We especially focus on visible focus indicators and discoverability. A high-contrast focus ring always shows exactly where you are on the page, so you never have to guess. A "Skip to main content" link at the very top lets you bypass the navigation bar instantly — press Tab once and then Enter to jump past the nav.

On the Things to Read page, all links are reachable by Tab and open with Enter. Arrow keys let you move between navigation items without cycling through the full tab order.
